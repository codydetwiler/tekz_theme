== Description ==
Tekz is a modern IT Solutions & Services WordPress Theme designed for IT companies, technology startups, software agencies, and related service providers. Built on Underscores, it offers full support for the WordPress block editor and advanced customization options.

== Installation ==
1. Upload the `Tekz` theme folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Themes' menu in WordPress.
3. Go to Appearance > Customize to configure your site settings.

== Frequently Asked Questions ==

= How can I customize the theme? =
You can customize the theme using the WordPress Customizer under Appearance > Customize. Adjust colors, menus, logos, and more to fit your brand.

= Is Tekz compatible with page builders? =
Yes, Tekz works seamlessly with popular page builders like Elementor and WPBakery.
